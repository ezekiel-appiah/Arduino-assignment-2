# Arduino-assignment-2
Arduino Uno project
This is my submission for assignment 2 in the programming C++ for Engineers at all Universities.
My project is a 7 - segment display counts down from 9 to 0 as well as a beep sound played by a buzzer.
# Hardwre used
- Arduino Uno
- 1 passive buzzer
- 1 single digit 7-segment display
- 220 ohms resister
- breadboard and jumper wires
## Concept demonstrated
-passive and active buzzers
-7 segment display wiring
-2d arrays for digital patterns
-Functions and parameters
-While and for loop
-Serial Monitor Output
## How to run it
Wire your buzzer to pin 8
wire the 7-segment display to pin 2 3 4 5 6 7 8 and 9
Open the .ino file in tghe Arduino IDE
Select tools > Board > Arduino Uno and the correct Port
Click Upload
